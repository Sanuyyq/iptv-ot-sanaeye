# 📺 IPTV Playlist Collection — iptv-ot-sanayeye

Набор IPTV-плейлистов из различных источников, собранных в одном репозитории для удобного использования, тестирования и резервного хранения.

Проект предназначен для регулярного обновления списков каналов и поддержания их работоспособности.

---

## 📂 Доступные плейлисты

В репозитории находятся следующие файлы:

* **tvteam.m3u8** — основной плейлист
* **ilook.m3u8** — альтернативный источник каналов
* **fawlok_iptv.m3u8** — архивный плейлист закрытого проекта
* **fgfgfdgdf.m3u** — тестовый список
* **neisvesno.m3u8** — дополнительный источник
* **neisvesmokto.m3u8** — экспериментальный список
* **brbrbr.txt** — вспомогательный файл

Файлы могут изменяться и обновляться по мере необходимости.

---

## 🚀 Использование

1. Открой нужный файл `.m3u` или `.m3u8`
2. Нажми кнопку **Raw**
3. Скопируй ссылку
4. Добавь её в IPTV-плеер

Пример ссылки:

```text
https://raw.githubusercontent.com/Sanuuyy/iptv-ot-sanayeye/main/tvteam.m3u8
```

---

## ⚙️ Совместимость

Плейлисты работают с большинством популярных IPTV-плееров:

* VLC
* OTT Navigator
* TiviMate
* IPTV Smarters
* Smart TV
* Android / Windows / Linux

---

## 🔄 Обновления

Списки каналов могут обновляться:

* при появлении новых источников
* при замене нерабочих потоков
* при исправлении структуры плейлистов
* при оптимизации названий каналов

Рекомендуется периодически обновлять список в плеере.

---

## ⚠️ Примечание

* Каналы могут временно не работать
* Источники могут изменяться без предупреждения
* Работоспособность зависит от внешних серверов
* Репозиторий не содержит собственных потоковых серверов
